# Projet Cube 3D Python

Ce projet affiche un cube 3D en rotation ("fil de fer") utilisant Python, Pygame et OpenGL.

## Prérequis

- Python 3.x installé.

## Installation

1.  Ouvrez un terminal dans le dossier du projet.
2.  Installez les dépendances nécessaires :

    ```bash
    pip install -r requirements.txt
    ```

## Lancement

Exécutez le script principal :

```bash
python main.py
```

Une fenêtre devrait s'ouvrir.
Maintenez le **clic gauche** de la souris et déplacez-la pour faire tourner le cube.
